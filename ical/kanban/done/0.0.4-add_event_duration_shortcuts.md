# Add Event Duration Shortcuts

Replace mandatory `--end` with duration shortcut flags on `ical add`.

## Changes

- `--short` → 15 minutes from start
- `--normal` → 30 minutes from start
- `--long` → 45 minutes from start
- `--end <datetime>` still accepted for explicit end times
- No flag provided → defaults to 30 minutes (same as `--normal`)
- Extracted `resolveEndDate(from:)` helper for testability
- Added `yyyy-MM-dd'T'HH:mm:ss` (timezone-less) format to `DateParser`
- 6 new unit tests in `AddEventCommandTests`
- Updated skill doc (`SKILL.md`)
