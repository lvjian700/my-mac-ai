# 1 — Bootstrap

Bootstrap a Swift 6.3 CLI app exposing Apple Calendar via EventKit.

## Decisions

- **No reminders** — calendar only
- **Flat command structure** — no `calendar` subgroup, commands live directly under `ical`
- **No delete** — deferred
- **macOS .v14 minimum** — required for `requestFullAccessToEvents()` async API
- **`main.swift` entry point** — avoids `@main`/`main.swift` conflict with ArgumentParser
- **`EventKitService` is `@MainActor`** — satisfies EventKit threading; `AsyncParsableCommand` auto-hops to main actor
- **`@preconcurrency import EventKit`** — suppresses Swift 6 Sendable warnings

## Commands

```
ical list      # list calendars
ical events    # list events (default subcommand)
ical add       # add event
```
