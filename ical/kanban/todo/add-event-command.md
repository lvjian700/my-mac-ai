# Add Event Command

Implement `ical add` (`AddEventCommand`) to create new calendar events.

## Arguments & Options

- `<title>` — event title (positional, required)
- `--start` — start date/time
- `--end` — end date/time
- `--calendar` — target calendar name or ID
- `--location` — event location
- `--notes` — event notes/description
- `--all-day` — flag to create an all-day event
