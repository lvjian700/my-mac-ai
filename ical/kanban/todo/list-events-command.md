# List Events Command

Implement `ical events` (`ListEventsCommand`) to query and display calendar events.

## Options

- `--from` — start date (supports "today", "tomorrow", ISO-8601, YYYY-MM-DD)
- `--to` — end date (same formats as `--from`)
- `--calendar` — filter by calendar name or ID
- `--format` — output format: `text` (default) or `json`
